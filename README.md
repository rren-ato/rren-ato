### Proyecto destacado

<details>
<summary><b>🧊 Liquid Glass Portfolio</b> — mi portfolio personal, hecho con React + Vite</summary>
<br>

Portfolio con estética "liquid glass" (vidrio líquido) construido desde cero, con foco en distorsión real vía filtros SVG (`filter: url()`, no `backdrop-filter` — mucho más confiable entre navegadores), partículas animadas, y un mini juego de air hockey jugable hecho a mano en canvas.

**Stack:** React · Vite · Canvas API · SVG Filters

**Repo:** [rren-ato/liquid-glass-portfolio](https://github.com/rren-ato/liquid-glass-portfolio)
